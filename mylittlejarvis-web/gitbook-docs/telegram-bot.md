# Telegram Bot

Use JARVIS through Telegram for mobile convenience.

## Setup

1. Create bot with [@BotFather](https://t.me/botfather)
2. Get bot token
3. Configure JARVIS:

```bash
jarvis config --set TELEGRAM_BOT_TOKEN=123456:ABC...
```

## Usage

Simply message your bot:

```
You: Hello, what can you do?
Bot: 🤖 I can help with coding, questions, 
     and tasks. What do you need?
```

## Features

- Same AI capabilities as CLI
- Mobile notifications
- Group chat support
- Voice message support

## CLI vs Telegram

| Feature | CLI | Telegram |
|---------|-----|------------|
| Speed | ⚡ Fast | 📱 Convenient |
| Privacy | 🔒 Local | ☁️ Cloud |
| Mobile | ❌ No | ✅ Yes |

Use both! CLI for development, Telegram for on-the-go.
