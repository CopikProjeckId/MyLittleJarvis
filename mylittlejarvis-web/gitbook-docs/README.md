# Introduction

**MyLittleJarvis** is an AI-powered assistant that runs directly in your terminal. Install with npm and start chatting with a powerful AI in seconds.

## What is JARVIS?

JARVIS transforms your terminal into an intelligent assistant capable of:

- **Natural conversations** - Chat with AI like talking to a person
- **Coding assistance** - Get help with programming tasks
- **Information retrieval** - Ask questions and get answers
- **Task automation** - Let AI handle repetitive tasks

## Quick Example

```bash
# Install
npm install -g my-little-jarvis-cli

# Start chatting
jarvis

# That's it! 🤖
```
