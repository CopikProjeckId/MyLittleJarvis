# Quick Start

Get your AI assistant running in just 3 minutes.

## Prerequisites

- Node.js 22+ 
- npm or yarn
- Terminal (macOS, Linux, or WSL on Windows)

## Install

```bash
npm install -g my-little-jarvis-cli
```

## First-Time Setup

Run the interactive wizard:

```bash
jarvis --setup
```

The wizard will guide you through:
1. Optional Telegram configuration
2. Optional Claude API key
3. Ollama local models (recommended)
4. Default persona selection

## Start Chatting

```bash
jarvis
```

You'll see:
```
🤖 JARVIS: Hello! How can I help you today?
> 
```

Type your message and press Enter!
