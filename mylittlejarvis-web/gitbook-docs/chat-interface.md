# Chat Interface

The JARVIS chat interface provides a powerful terminal-based conversation experience.

## Starting a Chat

Simply type `jarvis` in your terminal:

```bash
$ jarvis
🤖 JARVIS: Hello! How can I help you today?
> 
```

## Features

### Natural Language Input
Type your messages naturally. JARVIS understands context and maintains conversation history.

### Slash Commands

| Command | Description |
|---------|-------------|
| `/help` | Show available commands |
| `/reset` | Clear conversation history |
| `/persona [name]` | Switch AI persona |
| `/model [name]` | Switch AI model |
| `/exit` | Quit chat |

### Keyboard Shortcuts

- **↑/↓ arrows** - Navigate chat history
- **Ctrl+C** - Graceful exit
- **Enter** - Send message
- **Enter twice** - Multi-line input

### Code Blocks

JARVIS automatically formats code with syntax highlighting:

> "Show me a Python hello world"

```python
print("Hello, World!")
```

## Tips & Tricks

1. **Be specific** - Clear questions get better answers
2. **Use examples** - "Like this: ..."
3. **Iterate** - Refine based on responses
4. **Save sessions** - Use `/reset` to start fresh
