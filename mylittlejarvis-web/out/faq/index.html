Frequently Asked Questions

Everything you need to know about MyLittleJarvis. Can't find what you're looking for? Check the documentation or contact us.

What devices are supported?

+
MyLittleJarvis supports any Android phone running Android 8.0+ with at least 3GB RAM. We also support Raspberry Pi 4/5, and any Linux device that can run Docker or Python. iOS support is coming soon!

Do I need an internet connection?

+
Yes, for the initial setup and for cloud-based AI models. However, if you use local models via Ollama, you can run completely offline after setup. Some features like web search require internet connectivity.

Is my data private?

+
Absolutely! Your data never leaves your device unless you explicitly configure cloud integrations. We use local storage for memory, local AI models via Ollama for offline operation, and end-to-end encryption for any cloud communications.

How much does it cost?

+
MyLittleJarvis is completely free and open source! You only pay for API calls if you choose to use cloud-based AI models like Claude or GPT-4. Using local models with Ollama costs nothing beyond electricity.

Can I use it for coding?

+
Yes! JARVIS includes Claude Code integration, allowing you to code alongside a powerful AI assistant. It can read files, suggest improvements, debug errors, and even execute commands—all at a fraction of the cost of Claude Pro.
