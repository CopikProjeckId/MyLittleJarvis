Installation via Terminal

One command to rule them all. Your JARVIS awakens in minutes.

terminal — zsh
➜ git clone https://github.com/mylittlejarvis/jarvis-agent.git
➜ cd jarvis-agent && npm install
➜ cp .env.example .env
➜ ./setup.sh
➜ Scan QR code with your phone
➜ Start chatting with your JARVIS
➜ _

💡 Requires Node.js 18+ and Git. See documentation for detailed setup.
